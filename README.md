## Segmentación de vías aéreas pulmonares basado en machine learning

**Resumen:** Este trabajo aborda la segmentación automática de vías aéreas pulmonares en tomografías computarizadas (TC) como estrategia para optimizar el análisis morfométrico y el cálculo del biomarcador Total Airway Count (TAC), de relevancia clínica en enfermedades obstructivas como la EPOC. El propósito fue diseñar y evaluar un pipeline que integrara técnicas de segmentación mediante aprendizaje profundo y un esquema de esqueletización para el conteo estructurado de ramas bronquiales en sujetos sanos. Se utilizó la extensión MONAI Auto3DSeg de 3D Slicer, basada en la red SegResNetDS, aplicada sobre 30 TC con segmentaciones manuales de referencia provenientes de la base ATM'22. Posteriormente, se implementó un proceso de esqueletización con CGAL y análisis de datos en Python para obtener conteos por generación. El desempeño del modelo automático fue evaluado mediante métricas cuantitativas (Dice, IoU, precisión, sensibilidad y especificidad) y comparaciones con el gold standard por expertos. Los resultados evidenciaron un coeficiente de Dice promedio de 78.1\%, indicando una buena superposición global, pero con limitaciones en la detección de ramas distales, lo que ocasionó una subestimación significativa del TAC (62 \pm 19 frente a 197 \pm 52 manual). Este hallazgo muestra que, aunque el método es eficiente para la segmentación central, no es válido en su estado actual para la cuantificación clínica del TAC. Se concluye que la validez de los biomarcadores derivados depende de la integridad estructural de las segmentaciones, por lo que se recomienda emplear modelos entrenados específicamente para vías aéreas y complementar las métricas tradicionales con indicadores anatómicos.
